## Run 

파이썬 깔고 실행하기.

```bash
$ py main.py

    hello main()....

```

## 환경 설정

- [Anaconda 3.6](https://www.anaconda.com/download/#macos)를 우선 설치
- 설치 후 실행 환경 확인은 다음과 같이.

```bash
# after anaconda instance, check python version.
$ python -V

    Python 3.6.5 :: Anaconda, Inc.
    
```
